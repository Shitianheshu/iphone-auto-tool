# 🍎 iPhone自動購入ツール v1.3.3

## ✨ 新機能・更新内容

### 🔄 iPhone 17対応
- **URL更新**: iPhone 14 Pro → iPhone 17 に変更
- **通信キャリア**: 自動でSIMフリー選択（UNLOCKED_JP）
- **互換性**: iPhone 17の購入フローに対応

### 💻 マルチPC対応
- **4台のPC**: それぞれ別のスプレッドシートと連携
- **PC選択**: ドロップダウンでPC 1-4を選択
- **All View**: 全PCのスプレッドシートIDを同時表示・設定可能
- **個別スプレッドシート**: PCごとに異なるスプレッドシートIDを設定可能

## 🚀 使用方法

### 1. 基本設定
1. **購入ページURL**: iPhone 17の購入ページが自動設定
2. **PC選択**: 使用するPC番号を選択（1-4）
3. **スプレッドシートID**: 各PC用のスプレッドシートIDを設定

### 2. スプレッドシート設定
各PC用に以下のスプレッドシートを作成：
- **PC 1**: メインスプレッドシート
- **PC 2**: 2台目用スプレッドシート
- **PC 3**: 3台目用スプレッドシート  
- **PC 4**: 4台目用スプレッドシート

### 3. 購入設定
- **モデル**: iPhone 17のサイズ選択
- **カラー**: 希望するカラー選択
- **ストレージ**: 容量選択
- **受け取り方法**: 郵送/コンビニ/Apple Store
- **支払い方法**: クレジットカード/銀行振込

## 📋 スプレッドシート形式

以下の列が必要：
```
lastName, firstName, postalCode, state, city, street, street2, 
emailAddress, mobilePhone, cardNumber, expiration, securityCode, 
lockDatetime, status, Apple ID, Password
```

## ⚠️ 重要な注意事項

1. **通信キャリア**: 自動でSIMフリーが選択されます
2. **在庫監視**: 選択したストアの在庫を監視
3. **ロック機能**: 同時実行を防ぐためのロック機能
4. **エラーハンドリング**: 購入失敗時の自動リトライ

## 🔧 技術仕様

- **フレームワーク**: Electron + Puppeteer
- **言語**: JavaScript (Node.js)
- **API**: Google Sheets API
- **対応ブラウザ**: Chrome (Puppeteer)
